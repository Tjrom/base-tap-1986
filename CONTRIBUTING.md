# Contributing

Спасибо за интерес к проекту!

## Как внести изменения

1. Fork репозитория
2. Создай ветку для фичи (`git checkout -b feature/amazing-feature`)
3. Закоммить изменения (`git commit -m 'Add some amazing feature'`)
4. Запушь в ветку (`git push origin feature/amazing-feature`)
5. Открой Pull Request

## Правила коммитов

Используй понятные сообщения коммитов:
- `feat: добавить новую фичу`
- `fix: исправить баг`
- `docs: обновить документацию`
- `style: форматирование кода`
- `refactor: рефакторинг`

## Перед коммитом

Убедись, что:
- Код проходит `npm run lint`
- Проект собирается (`npm run build`)
- Локально всё работает (`npm run dev`)
